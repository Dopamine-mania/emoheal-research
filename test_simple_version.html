🎯 极简版音乐疗愈应用

✨ 应用简介 已简化
根据您的需求，我创建了一个极简版的音乐疗愈应用：
  • 去除了复杂的三阶段逻辑
  • 直接调用后端API获取疗愈视频
  • 简单清晰的用户界面
  • 输入情绪 → 获取视频 → 播放视频
🔄 工作流程
输入情绪文字 → AC情感分析 → KG音乐参数 → MI视频检索 → 播放视频
整个流程完全基于您已开发好的后端功能，前端只负责展示。
🚀 立即测试
打开极简版应用
📝 使用说明
  1. 点击"打开极简版应用"按钮
  2. 在文本框中输入您的情绪状态（如："头晕想吐"、"焦虑不安"等）
  3. 点击"开始疗愈"按钮
  4. 等待系统处理（可以看到进度条）
  5. 自动播放返回的疗愈视频
  6. 可以暂停/播放视频，或点击"结束疗愈"返回
⚠️ 注意事项：
  • 确保后端服务正在运行（端口5000）
  • 确保后端的视频文件路径配置正确
  • 如果视频无法播放，请检查后端的视频文件位置
